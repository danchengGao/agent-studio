## 安装方式

K8s 部署方式使用 Helm Chart 模板一键式部署 openJiuwen Studio 体验环境，仅支持在安装了 Linux 系统的 Kubernetes 集群上操作。

当前文档以 **华为云（RDS、DCS、CCE）** 为示例云厂商，演示如何在云上快速拉起一套可用的分布式体验环境。


## 前置要求

* 熟悉 Linux/Unix 系统操作
* 熟悉 云服务、云服务网络、容器化、Kubernetes、Helm 基本知识及操作
* 了解服务配置和管理
* 具备网络和代理配置经验
* 能够独立解决安装过程中的问题

## 需要安装的依赖服务

* MySQL 数据库
* Redis 数据库

> **注意**：本安装方式主要适合于想在分布式组网环境快速部署 Studio 体验环境。
