# Distributed Installation

## Installation Method

This installation method uses **Helm Charts** to deploy an openJiuwen Studio distributed experience environment on **Kubernetes**, and it is intended for **Linux-based Kubernetes clusters**.

This guide uses **Huawei Cloud (RDS, DCS, CCE)** as an example to help you quickly bring up a usable distributed environment in the cloud.

## Prerequisites

- Familiar with Linux/Unix operations
- Familiar with cloud services and networking, containerization, Kubernetes, and Helm basics
- Understand service configuration and operations
- Have experience with network/proxy configuration
- Able to troubleshoot common installation issues independently
- Suitable for users with professional operation and maintenance experience.

## Required Dependencies

- MySQL database
- Redis database

> Note: This installation method is mainly for users who want to quickly deploy a distributed Studio experience environment.
